Upcoming code sprints in Nashville and NYC

Rachel Reese    |     2015-08-25T10:01:00   |   community, f#

At Jet, we believe in community and giving back. We first heard about the F# Community Code Sprint idea a couple years ago, when they popped up in London. They expanded to NYC and to Nashville shortly afterwards, and we're ready to host a round of code sprints again in Nashville on August. 30 and in NYC on Sept. 12.

The idea of a "community code sprint" is to bring together active members of the community to a single place for a fun event where we can hack together on interesting projects, fix some of those long-standing bugs and feature requests, write amazing samples for talks, improve (or create!) documentation, and more.

Have you been wanting to contribute to F# open source, but aren't sure where to start? Check out the up-for-grabs issues that are tagged with F#!

Hosted on Azure, using FsBlog. Because F# + Azure |> Jet <3.
